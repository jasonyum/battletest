GenBoolean()

new GenBoolean()

Generates test data of data type "Boolean". this.ind tracks the number of times GenString() has been invoked. Each time GenString.next() is called, the class will output an object with "val" specifying the test value and "descript" specifying how the val was generated. Ex. { val: 5, descript: 'random intetger between 0 and 5' }.
